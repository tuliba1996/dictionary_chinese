# Panda Chinese

## Getting started

```
yarn install

# Development
yarn serve

# Build dist
yarn build

```

# What works?
### Front end
- Vue 3
- Composition API
- Vue router
- Vuex
- ESLint

### Backend 

- Python 3.8
- Django 
- Django Rest Framework
- SQLite3

# Functions

- Login (JWT)
- Register
- Dictionary Chinese
- Topic (Create, Delete)
- Profile
